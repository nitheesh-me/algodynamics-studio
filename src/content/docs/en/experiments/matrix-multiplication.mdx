## Mulitply and add Randomly

The Swap Machine is a simple machine that swaps two elements in an array. It is the basic building block for the Bubblesort algorithm.

## Order Machine

The Order Machine is a simple machine that compares two elements in an array and orders them in a specific way. It is the second building block for the Bubblesort algorithm.
